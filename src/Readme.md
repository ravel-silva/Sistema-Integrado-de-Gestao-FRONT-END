## 📖 Sobre o Projeto

Este repositório contém o front-end do **Sistema Integrado de Gestão**, responsável por fornecer uma interface visual para o gerenciamento de **funcionários**, **equipes** e **materiais** dentro de uma organização.

A aplicação consome uma [API desenvolvida em ASP.NET Core](https://github.com/ravel-silva/Sistema-Integrado-de-Gestao), possibilitando a realização de operações como cadastro, consulta, atualização e exclusão de dados.

---

## 🚀 Funcionalidades

✅ **Funcionários**  
- Cadastro, consulta, edição e exclusão  
- Associação com equipes  

✅ **Materiais**  
- Gerenciamento de materiais  
- Solicitações por equipes  

✅ **Equipes**  
- Criação e gerenciamento de equipes  
- Associação de funcionários  

✅ **Requisições de Material**  
- Solicitação de materiais por equipe  
- Histórico e status das requisições
